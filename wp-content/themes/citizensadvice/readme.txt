== Description ==
A starter WordPress theme created by Citizens Advice.


* Custom Header (home page featured image)
* Menus
* Widegts

== Introduction ==

This starter theme assumes you have a default WordPress installation with no additional plugins installed.
This starter theme allows you to install and host you own WordPress site that will be Citizens Advice branded.
This starter theme is provided without any support, however, if the theme is broken, please notify webteam@citizensadvice.org.uk


== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button, click the 'Upload theme' button
2. Browse and upload the Citizens Advice theme zip file.
3. Click on the 'Activate' button to use your new theme right away.

== Customization ==
1. Navigate to Appearance > Customize in your admin panel and customize
2. Citizens Advice theme contains 2 menu locations
	2.1 A Primary menu (blue bar)
	2.2 Header top navigation (top right, just above the main site search)
3. Homepage, is built by using Widgets, 8 sidebars/widgets in total
	3.1 Home 1 span 8/12 columns, it’s feature image is managed by the 'Header Image' menu within the Customization screen
	3.2 Home 2 is not used as the Citizens Advice site search is fixed to this position
	3.3 Home 3 - 7 span 4 columns individually
	3.4 Home 8 span 8 columns
4. Footer, 5 sidebars/widgets in total
	4.1 Footers 1 to 4 are set to span 3/12 columns
	4.2 Footer 5 is set to span 12 columns
5. Custom 404, A sidebar is available that will allow you to customise the message that appears on your 404 page.
